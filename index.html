Facetone User Creation Guide (HBL)

Follow the steps below to create a new agent user account for HBL.

Steps Overview

  1. Create User Account
  2. Create Resources
  3. Create SIP User Account
  4. Map User with SIP Account
  5. Add Scopes to the User

1. Create User Account

  1. Log into the admin console.
  2. Navigate to USER → Users.
  3. Fill in the user details:
    • Email: agent@hbl.com
    • Password: Hbl@123
    • User Type: Agent
    • First Name: Test
    • Last Name: Agent
    • Phone No.: use “1” if number is not provided
    • Nick Name: use the email address
  4. Click on Add User button. A notification will show success or error details.

2. Create Resources for the User

  1. Navigate to Request Router → Resources.
  2. Click on the + sign to create a new resource account.
  3. Type the user's email in Resource Name and press check.
  4. Insert Human, Agent, Call Center, Other Data and click Add.
  5. Edit resource account and set permissions:
    • Drag & drop Call task to access permission area.
    • Set Concurrency to 1.
  6. Assign Skills:
    • Drag skill from Available Skills to Assigned Skills.
    • Set the percentage for the skill and press Enter.
    • Repeat for all necessary skills.

3. Create SIP User Account

  1. Navigate to USERS → SIP Users.
  2. Find the last extension number and click New.
  3. Fill in details:
    • SIP Username: avoid special characters or numbers
    • Password: same as user account
    • Extension Number: next available
    • Pin: 1234
    • Email: user’s email address
    • Select domain & context, tick all fields
    • Deny Outbound: NONE
  4. Click Save. Notification will show success or error.

4. Map SIP User Account to User Account

  1. Navigate to USERS → User.
  2. Find the user and click the user icon.
  3. Click Edit Profile.
  4. Scroll down and select the SIP user account.
  5. Click Save Profile.

5. Add Scopes to the User

  1. Navigate to USERS → User and find the user.
  2. Click the settings icon.
  3. Drag & drop Agent Console from Access Denied to Access area.
  4. Reload browser (mandatory).
  5. Double click Agent Console, tick the fields, and click Update.
  6. Repeat for all scopes, one by one.

If there are any issues, contact DuoSoftware support team.
